# Risk of Ragnarok 2026.06.27

_Crystal Streak skill display_

- The **Basic Skill** entry in the skill window is now shown as **Crystal Streak**,
  reflecting how it actually works on Risk of Ragnarok.
- Updated the skill description to explain the streak (0–9), how it builds and
  resets, and its per-level bonuses:
  - +3% ATK & MATK per level (up to +27%), refreshed each floor
  - +75z per level on every floor kill
  - Per-kill chance for HD Oridecon (0.20%/lv) and HD Bradium (0.10%/lv)
  - Per-kill chance for a Morning Dew of Yggdrasil (0.05%/lv, max 0.45%)
  - Resets to 0 when you return to the Inn or die

Client-only display change (skill name + description). No gameplay values changed.
